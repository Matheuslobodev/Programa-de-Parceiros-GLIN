O que é a Glin?

Somos uma facilitadora de pagamentos internacionais que tem como missão democratizar o acesso dos brasileiros a produtos de qualidade das melhores marcas dos Estados Unidos.


Sabemos que redirecionadores e personal shoppers são fundamentais para esse acesso ser possível. Assim, além de giftcards das principais lojas dos EUA, oferecemos uma solução de pagamento para redirecionadores e personal shoppers.


Temos um time de tecnologia de primeira linha que trabalha incansavelmente para dar o melhor para os nossos clientes.
